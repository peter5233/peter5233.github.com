融资租赁中的工程设备远程监测


  • 解决方案
  • 主要规格
  • 客户案例

概述

企此实现设备的升级换代，首先要解决资金问题，对于高精尖的数控加工类设备，少则花费几百万元，多则花费上千万元甚至亿元，大型企业往往都要借助银行货款筹集资金，而中小型企业的融资渠道则更为有限。如何解决设备的资金问题，成为困扰生产加工型企业的难题。

在这种情况下，融资租赁是一种日益被企此接受的方式，融资租赁是一种以设备为载体，集贸易、金融、租借为一体的特殊金融产品，是由出租方融通资金为承租方提供所需设备，具有融资、融物双重职能的租赁交易。

在国际市场上，工程设备的融资租赁发展得如火如荼，在国内，融资租赁的业务和法律法规还不成熟，鉴于工程设备具有使用周期长，使用地点分散、管理难度大等特点，所以，对租赁设备进行风险控制和监督管理，成为融资租赁业务的重要组成部分。


现状

目前，要对设备实现远程监测，其普遍使用的是“GPS + GPRS”技术，其中，GPS负责定位，GPRS负责传输数据。

此类技术的核心逻辑，是定义了一个“安全地理范围”，即电子围栏，当被监测设备超出了合适的地理范围以外，则向出租方发出报警。但是，该技术的一个主要缺陷是，大部分情况下，设备是放在室内使用的，而室内一般没有GPS倌号，因此无法提供精准的定位信息。

另外，现有技术缺乏对设备的状态进行监测，比如设备的闲置、运行、出错等状态，若是出租方能够获取这些数据，分析出设备的利用率、空闲率、报错率、零件生产量等統计信息，则可以对融资租赁业务起到优化和改进作用。


新型监测系统

我们的设备租赁远程监测系统，结合了先进的射频技术、传感技术、物联网技术、数字通信技术、大数据技术和人工智能技术等，在传统的设备监测系统的基础上，增加了设备运行状态监非授权移动报警、拆卸报警、环境监测、工单调度管理、手机端微信通知等功能，克服了传统系统中设备安全管理力度不足、手段单一的弊病，提高了租赁设备监管的实时性、有效性、多样性，大幅改善了融资租赁交易中的风险控制水平。

该设备租赁远程监测系统中，包含了如下组件：

  • 多维度传感器，采集设备运行状态、地理位置、环境温湿度等信息；
  • GPRS/3G/4G/NB-IOT无线数据通信，无需布线，易于部署；
  • 云端大数据中心，负责数据分析、趋势监测、报警通知等功能；
  • 工单调度系统，方便人工检查、巡视工作的调度管理；
  • 微信和Web客户端，高效移动/互联网办公；

系统拓扑结构


特点

1. 丰富的感知报警

融合多种传感器，如GPS、振动\位移\加速度传感器、温度\湿度传感器，全方位监测设备安全。当遇到设备状态不良、非授权移动、或是工作环境有害等情况，系统及时予以报警。


2. 详细的设备报表

云端大数据中心，根据设备的状态监测，生成设备运行数据的日报/月报/年报，有利于分析设备的利用率、闲置率、报错率、零件生产量等統计信息。


3. 自动的工单管理

对于发生报警的设备，系统会自动为出租方发出检查工单，安排人工去现场勘察、排除异常。


4. 高效的移动办公

系统支持微信客户端数据展示，方便用户随时查看现场设备状态，和接收重要的消息推送。
